Verificar Estado de la API

Para conocer el estado de la API se debe agregar al dominio /ping

Se debe realizar un consumo de la API mediante GET. En caso de estar activa retorna un 200.

Uso del Modelo

Envía una imagen para el testeo del modelo y obtén unas de las 4 posibles clases:

  • Glioma
  • Meningioma
  • Pituitary
  • No_tumor

Se debe realizar un consumo de la API pasándole, mediante POST, un binario (imagen) con la key image. Para realizar este consumo se debe agregar al dominio /predict.

Las imágenes que acepta el modelo son png, jpg y jpeg.
